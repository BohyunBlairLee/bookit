# 디자인 리뉴얼 계획

## 디자인 분석 요약

### 홈 (빈 상태)
- 고양이 이모지, "읽고 있는 책이 없어요!" 텍스트
- 보라색 둥근 "책 추가하기" 버튼
- 헤더: 왼쪽 "홈", 오른쪽 "+" 아이콘

### 홈 (책 있을 때)
- "🔥읽는 중" + "1권/5권" 카운터
- 책 캐러셀: 커버, 제목, 작가, 출판사|날짜
- 진행률 바 (%) — 현재 없음, 추가 필요
- "독서 기록하기" 보라색 버튼

### 나의 책장
- 헤더: "나의 책장" + "+" 아이콘
- 필터 탭: 전체/읽는 중/읽을 예정/완독! (pill 스타일, 활성=보라)
- 3열 그리드: 커버 + 제목 + 작가

### 책 상세 (중요 변경!)
- 뒤로가기 화살표 (왼쪽 상단)
- "읽는 중 ▼" 드롭다운 (오른쪽 상단, 보라색)
- **수평 레이아웃**: 커버(왼), 제목+작가+출판사(오) — 현재는 세로 중앙 배치
- "독서 노트 +" 보라색 풀너비 둥근 버튼
- "📝독서 노트" 섹션 (회색 카드)
- **하단 네비게이션 보임** — 현재는 숨김

### 검색 (책 추가)
- 헤더: ← "책 추가" (중앙) 필터아이콘
- 검색창: 돋보기 + placeholder + X 클리어
- 검색 결과 선택 시 바텀시트: 상태 선택, 별점, 완독일, 달력
- "책 추가하기" 보라색 버튼

## 작업 목록
1. [x] 홈 화면 — 현재 거의 동일, 미세 조정
2. [ ] 책 상세 페이지 — 수평 레이아웃으로 변경, 하단 네비 표시
3. [ ] 나의 책장 — 미세 조정
4. [ ] 검색/책 추가 모달 — 헤더/검색창 스타일
5. [ ] 하단 네비게이션 — 책 상세에서도 표시
